# Changelog

All notable changes to ClockZone – World Time Calculator will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.2] - 2025-02-18

### Added
- Reverse time conversion feature (Local → Client)
- About page with version history dropdown
- Developer information and social links section
- Version badge in header
- Copyright footer with developer attribution
- Enhanced error handling for time conversions

### Changed
- Project name updated to "ClockZone – World Time Calculator"
- Improved timezone conversion accuracy
- Enhanced responsive design for mobile devices
- Updated UI/UX with better visual hierarchy

### Fixed
- Time conversion edge cases
- Mobile layout improvements
- Browser compatibility issues

## [2.1] - 2025-01-XX

### Added
- Bidirectional time converter system
- Forward conversion mode (Client → Local)
- Enhanced glassmorphism design effects
- 3D card tilt animations on hover
- Improved timezone auto-detection
- Persistent storage for user settings
- Enhanced modal interactions with animations

### Changed
- Redesigned converter section with toggle switch
- Improved card animations and transitions
- Enhanced color scheme and gradients

## [2.0] - 2024-12-XX

### Added
- Complete premium UI redesign
- World clock management system
- Real-time clock synchronization (updates every second)
- Display settings (12/24-hour format, date formats)
- Glassmorphism design system
- Smooth animations and micro-interactions
- LocalStorage integration for data persistence
- Add/remove world clocks functionality
- Timezone search and selection

### Changed
- Complete visual overhaul
- Modern dark theme with gradient accents
- Improved responsive layout

## [1.0] - 2024-11-XX

### Added
- Initial release
- Basic world clock display
- Simple timezone selection
- Basic responsive layout
- Core timezone functionality
